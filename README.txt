1)Для рендеринга модели открыть файл renderer_example.py и вызвать функицию rendering_ex('имя_файла.obj', 'имя_файла.mtl')
	При рендеринге образцовой модели в функции rendering_ex рекомендуется указывать папку сохранения рендеров "renders", а для проверяемой "renders1".
	!!!Рендеринг работает только с триангулрованными моделями. Для триангуляции рекомендуется использовать скрипт triang.py.
2)Для сравнения моделей открыть файл comare.py и вызвать одну из функций: rez("SSIM) для сравнения стандартным алгоритмом SSIM; rez("SSIMM) для модифицированного SSIM;  
	rez("MS-SSIM) для многомасштабного SSIM;  rez("G-SSIM) для градиентного SSIM.
3)Для вывода разницы между моделями открыть файл difference.py и запустить файл.
4)Для сбора статистических данных по набору моделей открыть файл stat.py. При необходимости создания текстового файла с оценками набора моделей вызвать функцию txtf("SSIM) для сравнения 
	стандартным алгоритмом SSIM; txtf("SSIMM) для модифицированного SSIM; txtf("MS-SSIM) для многомасштабного SSIM;  txtf("G-SSIM) для градиентного SSIM. При необходимости создания гистограммы
	вызвать функцию bar("алгоритм_сравнения"). Для создания круговой диаграммы вызвать функцию circle("алгоритм_сравнения"). Для создания диаграммы рассеяния вызвать функицию scat("алгоритм_сравнения").
	Создание диаграмм доступно только после создания текстовых файлов с оценками.
5)Для триангуляции модели открыть файл triang.py и вызвать функицию triang("имя_файла_obj", "имя_файла_mtl", "имя_выходного_файла_obj".
